## ⚠️ 常见问题

### ❌ 端口被占用

**错误信息**: `错误：端口 8000 已被占用！`

**解决方案**:
1. 关闭其他占用 8000 端口的程序
2. 或修改 `start_server.py` 中的端口号

### ❌ 数据库连接失败

**错误信息**: 数据库连接相关错误

**解决方案**:
1. 确保 MySQL 服务已启动
2. 检查 `static/.env` 文件中的数据库配置
3. 确认数据库 `note_app` 已创建

### ❌ 找不到配置文件

**错误信息**: `未找到配置文件`

**解决方案**:
确保 `_internal/static/.env` 文件存在并配置正确

## 🔐 配置说明

### 数据库配置

编辑 `_internal/static/.env` 文件：

```env
ASYNC_DATABASE_URL=mysql+aiomysql://root:你的密码@localhost:3306/note_app?charset=utf8
```

### AI 配置（可选）

```env
LLM_MODEL_ID="qwen-long-latest"
LLM_API_KEY="你的 API 密钥"
LLM_BASE_URL="https://dashscope.aliyuncs.com/compatible-mode/v1"
```

## 💡 最佳实践

### ✅ 推荐做法

1. **将整个文件夹复制到桌面** - 方便管理
2. **创建桌面快捷方式** - 一键启动
3. **保持窗口最小化** - 服务在后台运行
4. **定期重启服务** - 释放内存

### ❌ 不推荐做法

1. 不要删除 `_internal` 文件夹 - 包含必要的依赖
2. 不要移动单个文件 - 保持文件夹结构完整
3. 不要同时运行多个实例 - 会导致端口冲突
